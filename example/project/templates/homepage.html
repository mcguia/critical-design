{% trans "The demo features" %}:

  1. {% blocktrans %}jQuery plugin jquery.django_comments_xtd.js for Ajax interactions.{% endblocktrans %}
  2. {% trans "An Articles App, with a model Article. Article instances accept comments." %}
  3. {% blocktrans %}COMMENTS_XTD_MAX_THREAD_LEVEL set to 2. Meaning that comments can be nested up to a 2nd level:{% endblocktrans %} 
    First comment (level 0)
    |-- Reply to First comment (level 1)
        |-- Reply to Reply to First comment (level 2)
  4. {% blocktrans %}Comments have to be confirmed by email, or otherwise set COMMENTS_XTD_CONFIRM_EMAIL to False.{% endblocktrans %}
  5. {% blocktrans %}It uses the template tag render_markup_comment to render comments content. So you can use line breaks, markdown or reStructuredText to format comments{% endblocktrans %}. To use special formatting, start the comment with the line: #!<markup-lang> being <markup-lang> any of the following:
    • markdown
    • restructuredtext
    • linebreaks
  6. It uses django-crispy-forms to quickly provide Twitter-bootstrap3 integration.
